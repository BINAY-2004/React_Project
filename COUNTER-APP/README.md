# 🚀 React Counter App

A simple and interactive Counter App built using **React**.  
This project focuses on understanding core React concepts like **useState**, **state updates**, and **UI styling with inline CSS**.

## ✨ Features

- 🔢 Increase & Decrease Counter
- 🎯 State management using `useState`
- 🎨 Clean UI with inline CSS styling
- ⚡ Smooth animation on counter update
- 🧠 Demonstrates React state batching & functional updates

---

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6+)
- HTML5
- Inline CSS
